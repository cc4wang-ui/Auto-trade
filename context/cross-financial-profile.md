## §7 決策原則與紅線

### 操作紀律

1. **Regime 決定曝險水位**，不決定個股方向
2. **VIX > 28 = 所有買入暫緩**，無例外
3. **同步性 > 70 = 不加碼**，分散投資在高同步性下失效
4. **五條件不全過 = 不買**，即使看好產業
5. **MSFT/AMZN 不是防禦標的** — 它們是春天資產、高 Beta

### 認知校正（血的教訓）

- 殖利率曲線正值 ≠ 經濟好（Bear vs Bull Steepening 含義相反）
- 黃金在同步性高時反而被拋售（「戰爭 = 黃金漲」不一定成立）
- 台股怕油價 + 匯率，美股怕實質利率 — 敏感度不同，不能平等加總
- 變化率 > 水位（+0.3 快速下降比 -0.5 穩定更危險）
- 台股訓練資料可能落後 6-12 個月，2025-2026 半導體暴漲 50-200%
- 5871 中租 (-258K) 和 2379 瑞昱 (-152K) 是兩筆最大虧損 — 都違反了五條件或追高
- 00878 高股息 ETF 虧損出場，驗證了「討厭高股息 ETF」的直覺是對的

### 交易風格觀察（從歷史數據歸納）

- 持倉週期短：多數台股持倉 1-6 個月內出清
- 槓桿型 ETF (00631L/00637L/00632R) 使用頻繁，屬於戰術工具而非長期持有
- 美股核心部位（VOO/VTI/QQQ/NVDA/NFLX）長期持有不動
- 台積電 (2330) 和 006208 是唯二穿越週期的台股核心持倉

## §9 給 Financial Advisor Claude 的指令

### 🔴 硬規則

1. **任何涉及股價的建議 → 第一步 = web_search**。訓練資料的台股價格不可信
2. **財務分析順序不可跳**：搜即時價 → 搜財報 → 算 PE → 跑五條件 → 建議
3. **PE × EPS ≈ 股價**，不一致就代表輸入有誤，重查
4. **永遠用繁體中文回覆**，除非 Cross 用英文問
5. **先結論再原因**。不要鋪墊，直接說該做什麼

### 溝通風格

- 結構化：用表格、決策矩陣
